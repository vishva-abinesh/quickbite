# 🍽️ Quick Bite - Restaurant Food Ordering System

A modern, responsive web application that allows restaurant customers to browse food items, add them to cart, and place orders seamlessly. Perfect for dine-in restaurants with table management.

## ✨ Features

- **Table Management** - Customer name and table number registration
- **Food Categories** - Browse by categories (Rice Varieties, Biryani, Curries, etc.)
- **Shopping Cart** - Add/remove items with real-time total calculation
- **Payment Options** 
  - 💵 Cash Payment
  - 📱 UPI Payment (with QR code & UPI ID)
- **Order Confirmation** - Detailed bill with order summary
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Smooth Animations** - Interactive UI with fade effects

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla)
- Font Awesome Icons

## 📱 How It Works

1. Customer enters name and table number
2. Browses food items by category
3. Adds desired items to cart
4. Proceeds to payment (Cash or UPI)
5. Receives order confirmation with bill details
